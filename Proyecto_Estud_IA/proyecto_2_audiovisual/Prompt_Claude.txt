REGLA 1 — CERO LaTeX:
Nunca uses MathTex() ni Tex() en ninguna parte del código.
Usa ÚNICAMENTE Text() para todo texto y fórmulas.
Razón: MathTex llama a LaTeX, que congela Colab mostrando
"Writing..." sin terminar nunca. Esto es innegociable.

REGLA 2 — AUDIO con Google Gemini TTS:
La función de síntesis de voz es exactamente esta firma:

    def retornar_MP3(texto, api_key, nombre_archivo,
                     carpeta=CARPETA_AUDIOS, forzar=False,
                     max_reintentos=4, espera_inicial=3.0) -> dict:

Recibe texto y nombre_archivo, guarda en Drive Y en /content/,
retorna dict con claves: bytes, ruta_drive, ruta_local,
duracion, desde_cache.
Si el archivo ya existe en Drive lo reutiliza sin llamar a la API.

REGLA 3 — RATE LIMIT de la API (tier gratuito: 3 req/min):
- Pausa proactiva de 90s cada 3 requests nuevos.
- Pausa de 5s entre requests del mismo lote.
- Si llega un 429: reintento con espera 90s, 120s, 150s...
- NO contar como request los archivos del caché de Drive.
- Mostrar countdown visible durante cada espera.

REGLA 4 — DRIVE como almacenamiento permanente:
- Carpeta fija: /content/drive/MyDrive/manim_audios/
- Cada MP3 se guarda en Drive Y en /content/ (para Manim).
- Al recuperar de Drive: copiar a /content/ con shutil.copy().
- Exportar DURACIONES a duraciones.json antes de renderizar.
- Esto garantiza que los audios sobrevivan reinicios del runtime.

REGLA 5 — SINCRONIZACIÓN audio/video por escena:
Cada escena debe:
a) Llamar self.add_sound("nombre_audio.mp3") al inicio.
b) Calcular ANIM_TOTAL = suma exacta de todos los run_time.
c) Terminar con:
   self.wait(max(0.3, dur("nombre_audio.mp3") - ANIM_TOTAL))
   para que audio y video terminen exactamente juntos.

REGLA 6 — DISPLAY automático del video:
La celda de render usa -qm (720p mínimo) e incluye al final
el bloque base64 para que el video aparezca automáticamente
sin ejecutar celdas adicionales.

REGLA 7 — ESTRUCTURA de celdas (exactamente 5):
- CELDA 0: Montar Drive + crear carpeta de audios.
- CELDA 1: Instalación de dependencias + función retornar_MP3.
- CELDA 2: Lista GUIONES + generación con control de rate limit.
- CELDA 3: %%writefile animacion.py con el código Manim completo.
- CELDA 4: !manim -qm + display automático del video.

PRINCIPIOS DE DISEÑO (no negociables):

1. PALETA COHERENTE
   - Elige UNA familia de color dominante según el tema

   - Puntos de referencia y elementos clave: color brillante

2. TIPOGRAFÍA
   - Solo Text() de Manim. Cero MathTex(). Cero LaTeX.
   - Jerarquía de 3 niveles:

       · Medio  (22-38px): ecuaciones y subtítulos.

   - Peso BOLD solo para el título del caso.
   - Todo lo demás en peso normal.

3. LAYOUT FIJO (no cambiar nunca)
   - Gráfica o plano: move_to(LEFT * 2.8)
   - Panel informativo: move_to(RIGHT * 4.8)
   - Línea divisora sutil entre ambos lados.
   - El panel nunca tapa la gráfica.

4. PANEL INFORMATIVO (estructura fija, colores libres)

   - Barra superior sólida del color del caso.

   - Título del caso en grande y en color del caso.
   - Subtítulo descriptivo en blanco.
   - Tres secciones separadas por líneas horizontales:
       · Sección FUNCIÓN: micro-label + ecuación en color.
       · Sección REGLA:   micro-label + regla en gris claro.
       · Sección PUNTO:   micro-label + nueva posición en amarillo.

6. ANIMACIONES con personalidad:

   - Puntos clave: FadeIn(scale=1.8).

7. INTRO con identidad visual:

def retornar_MP3(texto, api_key, nombre_archivo,
                 carpeta=CARPETA_AUDIOS, forzar=False,
                 max_reintentos=4, espera_inicial=3.0):

    ruta_drive = os.path.join(carpeta, nombre_archivo)
    ruta_local = f"/content/{nombre_archivo}"

    if os.path.exists(ruta_drive) and not forzar:
        shutil.copy(ruta_drive, ruta_local)
        seg = AudioSegment.from_file(ruta_drive, format="mp3")
        dur = len(seg) / 1000.0
        print(f"   ♻️  Reutilizado: {nombre_archivo} ({dur:.2f}s)")
        with open(ruta_local, "rb") as f:
            mp3_bytes = f.read()
        return {"bytes": mp3_bytes, "ruta_drive": ruta_drive,
                "ruta_local": ruta_local, "duracion": dur,
                "desde_cache": True}

    print(f"   🎙️  Generando: {nombre_archivo}")
    client = genai.Client(api_key=api_key)
    ESTILO = ("Habla con voz profesional, pausada, muy emotiva "
              "y con acento paisa de Medellín, Colombia. "
              "Texto a leer:\n\n")
    config_tts = types.GenerateContentConfig(
        response_modalities=["AUDIO"],
        speech_config=types.SpeechConfig(
            voice_config=types.VoiceConfig(
                prebuilt_voice_config=types.PrebuiltVoiceConfig(
                    voice_name="Puck"))))

    response = None
    for intento in range(1, max_reintentos + 1):
        try:
            print(f"      🔄 Intento {intento}/{max_reintentos}...")
            response = client.models.generate_content(
                model="gemini-2.5-flash-preview-tts",
                contents=ESTILO + texto,
                config=config_tts)
            print(f"      ✅ Respuesta recibida.")
            break
        except Exception as e:
            msg = str(e)
            es_transitorio = any(
                c in msg for c in ["500","503","INTERNAL","UNAVAILABLE"])
            if es_transitorio and intento < max_reintentos:
                espera = espera_inicial * (2 ** (intento - 1))
                print(f"      ⚠️  Error. Esperando {espera:.0f}s...")
                time.sleep(espera)
            else:
                raise RuntimeError(
                    f"❌ Falló tras {intento} intento(s): {e}") from e

    try:
        audio_data = (response.candidates[0]
                              .content.parts[0]
                              .inline_data.data)
        assert audio_data
    except (AttributeError, IndexError, AssertionError) as e:
        raise RuntimeError(f"❌ Sin audio: {e}") from e

    wav_buffer = io.BytesIO()
    with wave.open(wav_buffer, "wb") as wf:
        wf.setnchannels(1)
        wf.setsampwidth(2)
        wf.setframerate(24000)
        wf.writeframes(audio_data)
    wav_buffer.seek(0)

    mp3_buffer = io.BytesIO()
    AudioSegment.from_wav(wav_buffer).export(
        mp3_buffer, format="mp3", bitrate="192k")
    mp3_bytes = mp3_buffer.getvalue()

    with open(ruta_drive, "wb") as f: f.write(mp3_bytes)
    with open(ruta_local, "wb") as f: f.write(mp3_bytes)

    dur = len(AudioSegment.from_file(
        io.BytesIO(mp3_bytes), format="mp3")) / 1000.0
    print(f"      💾 Drive: {ruta_drive}  |  {dur:.2f}s")
    return {"bytes": mp3_bytes, "ruta_drive": ruta_drive,
            "ruta_local": ruta_local, "duracion": dur,
            "desde_cache": False}
