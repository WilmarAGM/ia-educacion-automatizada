# Automatización Educativa con Inteligencia Artificial

Este repositorio contiene dos proyectos desarrollados en el contexto de un curso de inteligencia artificial.

## 📌 Proyecto 1: Generación Automática de Exámenes

Sistema que:
- Genera múltiples versiones de ejercicios
- Calcula soluciones automáticamente
- Permite evaluación masiva

🔗 Notebook: proyecto_1_examenes/notebook_examenes.ipynb  
🔗 Prompt: proyecto_1_examenes/prompt_chatgpt.txt  

---

## 📌 Proyecto 2: Generación de Contenido Audiovisual

Pipeline completo:
imagen → solución → guion → audio → animación → video

Tecnologías:
- Claude Sonnet 4.6
- Google Gemini TTS
- Manim

🔗 Notebook: proyecto_2_audiovisual/notebook_manim.ipynb  
🔗 Prompt: proyecto_2_audiovisual/prompt_claude.txt
